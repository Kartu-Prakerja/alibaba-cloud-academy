Tentang Program Ini

Program beasiswa ini adalah hasil kolaborasi Prakerja dan Alibaba Cloud Academy (ACA) untuk membangun talenta digital yang lebih berkualitas di Indonesia, terutama dalam keahlian cloud technology.

Terdapat 2 pilihan program, yaitu Cloud Computing dan Big Data. Program pelatihan dan sertifikasi berdurasi 1 bulan.

Alibaba Cloud Academy sendiri adalah bagian dari Alibaba Cloud yang berfokus dalam memberikan program pelatihan dan sertifikasi teknologi Alibaba Cloud.

Pilihan Program

Cloud Computing

Di program ini, peserta mempelajari hal-hal mendasar Alibaba Cloud, berikut konsep, dan produknya. Materi akan mencakup cloud computing, keamanan, jaringan, penyimpanan, dan basis data (database).

Service Review
Elastic Compute Service (ECS)
Virtual Private Cloud (VPC)
Server Load Balancer (SLB)
Object Storage Service (OSS)
Relational Database Service (RDS)

Big Data

Di program ini, peserta mempelajari komponen-komponen penting dari ekosistem Big Data dalam Alibaba Cloud. Materi mencakup DataWorks, MaxCompute, QuickBI, DataV, dan bagaimana teknologi-teknologi ini mendukung satu dengan yang lainnya.
